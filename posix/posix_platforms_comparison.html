mmap(), munmap()
signal(), sigaction()
sem_open(), sem_wait()
Implementación: glibc
Conformidad POSIX completa (IEEE 1003.1)
Extensiones GNU adicionales
Threading robusto con NPTL
Filesystem POSIX completo
Signals avanzados (real-time)

POSIX APIs Disponibles

fork(), exec()
open(), read(), write()
socket(), bind()
mmap(), munmap()
signal(), sigaction()
dispatch_queue (ext)
Implementación: libSystem
Certificación POSIX oficial (SUSv3)
BSD Unix base (Darwin kernel)
Grand Central Dispatch (GCD)
Mach microkernel underneath
Framework-specific extensions
Android

POSIX APIs Disponibles

pthread_create()
pthread_mutex_lock()
open(), read(), write()
socket(), bind()
mmap(), munmap()
signal() (limitado)
sem_open() (limitado)
fork() (no disponible)
Implementación: Bionic libc
Subset POSIX optimizado para móviles
No fork() - solo threads
Startup optimizado
Memory footprint reducido
Thread-local storage optimizado
