4. Métricas On-Premise

Se probaron tres configuraciones diferentes para determinar la relación óptima entre calidad, tiempo de procesamiento y costo energético.

Configuración Resolución Tiempo Energía Costo
Config 1 1280×704 513s (8.5min) 0.0264 kWh S/0.017
Config 2 768×768 198s (3.3min) 0.00274 kWh S/0.0018
Config 3 ★ 1280×720 198s (3.3min) 0.00274 kWh S/0.0018

Config 3: Mejor Opción

  • 89% más económico que Config 1
  • Resolución HD completa (1280×720)
  • 90% reducción de energía vs Config 1
  • Duración de video: ~1.7 segundos

Datos del Hardware

  • GPU: NVIDIA RTX 5070
  • Tarifa eléctrica: S/0.62/kWh (promedio 2024-2026)
  • Medición: Wattímetro en tiempo real
  • Software: ComfyUI + Modelo WAN

6. Comparativa On-Premise vs Cloud

On-Premise

S/0.0018
por video (~1.7s)

Ventajas

  • • Control total del proceso
  • • Costos extremadamente bajos
  • • Sin límites de uso
  • • Privacidad garantizada
  • • Personalización completa

Desventajas

  • • Inversión inicial en hardware
  • • Conocimiento técnico requerido
  • • Mantenimiento propio

Cloud Services

$3-5
por video corto

Ventajas

  • • Sin inversión inicial
  • • Fácil de usar (API/UI)
  • • Calidad consistente
  • • Soporte técnico incluido
  • • Actualizaciones automáticas

Desventajas

  • • Costos recurrentes altos
  • • Límites por plan/créditos
  • • Menos control creativo

8. Conclusiones y Recomendaciones

Hallazgos Clave

  • • La solución on-premise es 5,000-8,000x más económica que servicios cloud
  • • Inversión inicial se recupera en 1-2 meses con producción moderada
  • • Workflow validado y reproducible para contenido de calidad media-alta
  • • Config 3 (1280×720, S/0.0018/video) identificada como configuración óptima

Recomendaciones

  • • Implementar solución on-premise para producción regular
  • • Optimizar workflow para reducir tiempo de generación
  • • Explorar modelos adicionales (Stable Diffusion, LCM)
  • • Documentar mejores prácticas y crear templates
  • • Escalar gradualmente: 200-500 videos/mes en corto plazo

Próximos Pasos

  • Corto plazo (1-2 meses): Escalar a 200-500 videos/mes
  • Mediano plazo (3-6 meses): Automatización completa del pipeline
  • Largo plazo (6-12 meses): Desarrollo de SaaS basado en PoC

Oportunidades de Negocio

Generación de contenido para redes sociales, producción de material educativo, contenido para e-commerce, videos explicativos automatizados y personalización masiva de contenido.
